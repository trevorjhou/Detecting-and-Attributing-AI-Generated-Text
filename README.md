## Project Overview

As large language models become increasingly capable, organizations need reliable ways to determine whether a piece of text was written by a human or generated by AI.

This project investigates two related tasks:

### 1. Detection (Binary Classification)

Determine whether a document was written by:

* Human
* AI

### 2. Attribution (Multiclass Classification)

Determine which AI model generated a document.

The attribution task includes six frontier models from Anthropic and OpenAI.

---

## Research Questions

1. Can we accurately distinguish human-written text from AI-generated text?

2. Can we identify which frontier AI model generated a document?

3. Which modeling approaches perform best:

   * Classical Machine Learning
   * Tree-Based Models
   * Neural Networks
   * Transformer Models

#### Anthropic

#### OpenAI

Each AI document was generated from the same source prompt as its corresponding human document, creating a controlled benchmark for fair comparison.

## Modeling Pipeline

```text
Raw Text
    ↓
TF-IDF Vectorization
    ↓
SVD Dimensionality Reduction
(20,000+ → 300 Features)
    ↓
Model Training
    ↓
Evaluation
```

### Feature Engineering

#### TF-IDF

Converts text into weighted word features.

Words that are distinctive within a document receive higher weights, while common words receive lower weights.

#### SVD

Reduces approximately 20,000 TF-IDF features into 300 latent dimensions while preserving most of the important information.

## Experimental Design

### Leakage Prevention

A grouped train-test split was used based on source prompts.

This prevents the same prompt from appearing in both training and testing sets, ensuring that models learn writing patterns rather than memorizing prompt content.

### Hyperparameter Tuning

Neural network models were tuned using:

* Optuna
* Early Stopping
* Validation-Based Model Selection

## Key Findings

### Detection Is Nearly Solved

AI-generated text can be identified with approximately 98% accuracy on this benchmark.

### Attribution Remains Difficult

Attribution accuracy drops to approximately 64%, demonstrating that identifying which AI generated a document is substantially harder than detecting AI-generated text itself.

### Complexity ≠ Better Performance

The simplest model tested, Logistic Regression, outperformed:

* LightGBM
* CNN
* LSTM
* SimpleRNN
* DistilBERT
* Ensemble Models

More complex models did not consistently provide better results.

### Attribution Errors Are Systematic

Most attribution errors occurred within the same vendor family.

OpenAI Flagship, Mid, and Mini models were frequently confused with one another, suggesting highly similar writing styles.

## Main Takeaway

The most important finding from this project is:

> Detecting AI-generated text is relatively easy, but identifying which AI generated the text remains a significantly more challenging problem.

Furthermore, sophisticated models did not outperform simpler approaches, demonstrating the importance of matching model complexity to the structure of the problem.
